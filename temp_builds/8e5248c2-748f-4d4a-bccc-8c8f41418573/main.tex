\answeredbox{式 $x^3 - 3x^2 - 4x + 12$ を因数分解します。\n\nまず、因数定理を用いて $x = 2$ が方程式の解であることを確認します。\n$$2^3 - 3(2)^2 - 4(2) + 12 = 8 - 12 - 8 + 12 = 0$$\n\nよって、$x - 2$ は因数です。\n\n次に、$x^3 - 3x^2 - 4x + 12$ を $x - 2$ で割ります。\n割り算を行い、商は $x^2 - x - 6$ になります。\n\n$x^2 - x - 6$ は、さらに $(x + 2)(x - 3)$ に因数分解できます。\n\nしたがって、$x^3 - 3x^2 - 4x + 12 = (x - 2)(x + 2)(x - 3)$ です。}
\end{qbox}
