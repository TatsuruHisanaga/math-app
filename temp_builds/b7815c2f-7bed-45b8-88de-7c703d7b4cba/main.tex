\answeredbox{直角三角形の斜辺の長さはピタゴラスの定理を使って求められます。\nピタゴラスの定理は $a^2 + b^2 = c^2$ で表され、$a$ と $b$ は直角を挟む辺の長さ、$c$ は斜辺の長さです。\nここで、$a = 3$ cm, $b = 4$ cm なので、$c = \sqrt{3^2 + 4^2} = \sqrt{9 + 16} = \sqrt{25} = 5$ cm です。}
