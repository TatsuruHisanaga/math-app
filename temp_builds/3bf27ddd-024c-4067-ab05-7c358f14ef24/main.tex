\answeredbox{花壇の周囲の長さが$100$メートルなので、$2x + 2y = 100$です。これを簡単にすると、$x + y = 50$となります。\n面積を最大にするための方程式は、$A = xy$ですが、$y = 50 - x$を代入すると、$A = x(50 - x) = -x^2 + 50x$です。この2次関数の頂点を求めると、\[ x = -\frac{b}{2a} = -\frac{50}{2(-1)} = 25 \] よって、$x = 25$のとき面積が最大で、$y = 50 - 25 = 25$です。}
